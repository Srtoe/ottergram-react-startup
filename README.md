# Ottergram React Startup Page

This project was created for the Week 1 Create React App assignment. It uses a Create React App style folder structure and includes basic routing with `react-router-dom`.

## How to run the project

1. Install Node.js if it is not already installed.
2. Open a terminal in this project folder.
3. Run `npm install`.
4. Run `npm start`.
5. Open `http://localhost:3000` in a browser if it does not open automatically.

## Pages included

- Home: startup page and assignment goal
- Gallery: placeholder Ottergram cards
- About: project description and future improvements

## Notes

The assignment manual uses `Switch` and `Route`, so this project uses `react-router-dom` version 5.3.4 to match that syntax. Newer React Router versions use `Routes` instead of `Switch`.
